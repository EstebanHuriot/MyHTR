# MyHTR
Handwritten Text Recognition so I can write my code instead of typing it.

## Structure of the project
MyHTR/
│
├── app.py          # FastAPI API
├── func.py         # Handling models
│
├── index.html      # User interface
├── scripts.js      # Managing the canva and the api requests
├── style.css       # Nice looking user interface
│
└── README.md

## Models
Currently using TrOCR (see https://huggingface.co/microsoft/trocr-base-handwritten).
